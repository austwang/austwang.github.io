Video Games

  1. NES Metroid in Unity

    This game is a part of Metroid (NES) remade in Unity.

    This two-person project was mainly done in Unity with the C# language for scripts.

    I contributed to implementing the main movement mechanics and creation of enemies.

    I also helped add in the game audio and experimented with creating animations, with Samus being the most complicated so far.

  2. Zipline Jaunt

    This project was mainly done in Unity with the C# language for scripts.

    This is a prototype 2D game where you're able to deploy traversible ziplines.

    The player is able to see where they can place the zipline, as well as adjusting its length and angle.

  3. Just Jump

    This is a fast-paced 2D platformer game where you must quickly climb your way up levels and avoiding various obstacles both from above and below.

    This game was created in Unity and C# with a team of five developers.

    My contribution is that I helped create much of the UI in the game such as the main menu and pause menu.

    I also worked on the scoring system and leaderboard system that a player can see at the end of each level.
